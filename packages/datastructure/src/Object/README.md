# Object
- 객체는 키-값 쌍의 정렬되지 않은 컬렉션입니다. 키는 문자열 또는 Symbol 타입이어야 하며 값은 모든 데이터 타입이 될 수 있습니다.
- 값을 검색하려면 해당 키를 사용할 수 있습니다. 이것은 점 표기법 또는 대괄호 표기법을 사용하여 달성할 수 있습니다.
- 객체는 반복 가능하지 않습니다. `for...of`와 함께 사용할 수 없습니다.

# Time Complexity
- Insert - O(1)
- Remove - O(1)
- Access - O(1)
- Search - O(n)
- Object.keys() - O(n)
- Object.values() - O(n)
- Object.entries() - O(n)
